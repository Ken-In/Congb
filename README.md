# Congb
Congb Engine

参考项目：https://github.com/TheCherno/Hazel

该项目使用vs2022编译，只支持Windows。

双击运行项目目录下GenerateProject.bat即可生成项目。

- 添加EntryPoint
- 添加Log
- 添加premake来生成项目
- 添加EventSystem
- 添加预编译头文件PCH
- 添加Window抽象和GLFW
- 添加WindowsWindow GLFW Callback
- 添加Layer
- 添加Glad
- 添加ImGui
- 添加事件到ImGui
- 添加Input
- 添加glm
- 添加imgui docking 修改 imguiLayer
- 绘制三角形
