# Congb
Congb Engine

闲着无聊就撸个引擎吧！

该项目使用vs2022编译，暂时只支持Windows。

点击GenerateProject.bat就可以生成项目，把SandBox设为启动项目，因为Congb.dll需要拷贝到SandBox目录下，第一次运行会报错，第二次运行就没问题了。

- 添加EntryPoint
- 添加Log
- 添加premake来生成项目
- 添加EventSystem
- 添加预编译头文件PCH
- 添加Window抽象和GLFW
- 添加WindowsWindow GLFW Callback
- 添加Layer
- 添加Glad
- 添加ImGui
